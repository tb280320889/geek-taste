<interfaces>
auth.rs Tauri command 模式：
```rust
#[tauri::command]
pub async fn validate_github_token(token: String) -> Result<ValidateTokenResponse, String> {
    match github_adapter::auth::validate_token(&token).await {
        Ok(user) => Ok(ValidateTokenResponse { success: true, user: Some(UserDto::from(user)), error: None }),
        Err(e) => Ok(ValidateTokenResponse { success: false, user: None, error: Some(e.to_string()) }),
    }
}
```

    2. 创建 crates/application/src/topk.rs，实现以下用例：

    a) `execute_ranking(conn, token, budget, view_id) -> Result<Vec<RankingItemDto>>`:
       - 从 DB 获取 RankingView
       - 根据 filters 构建 SearchQuery
       - 调用 github_adapter::search::search_repositories 获取候选集
       - 将结果 upsert 到 repositories 表
       - 根据 RankingMode 排序：
         - STARS_DESC: 按 stargazers_count DESC
         - UPDATED_DESC: 按 updated_at DESC
         - MOMENTUM_*: 获取上次 snapshot，计算 MomentumScore，按 total DESC
         - 若无历史 snapshot 且是 MOMENTUM 模式：降级为 UPDATED_DESC
       - 截取前 k_value 条
       - 转换为 RankingItemDto（含 score_breakdown for momentum）
       - 返回结果

    b) `list_views(conn) -> Result<Vec<RankingViewSpecDto>>`:
       - 调用 persistence_sqlite::ranking_repository::list_ranking_views
       - 转换为 RankingViewSpecDto

    c) `create_view(conn, request: CreateRankingViewRequest) -> Result<RankingViewSpecDto>`:
       - 构建 RankingView（生成 ULID id，设置 created_at/updated_at）
       - 调用 persistence_sqlite 保存
       - 立即触发暖机快照（在后台或同步）
       - 返回 RankingViewSpecDto

    d) `delete_view(conn, view_id) -> Result<()>`:
       - 调用 persistence_sqlite::ranking_repository::delete_ranking_view

    e) `toggle_pin_view(conn, view_id) -> Result<()>`:
       - 调用 persistence_sqlite::ranking_repository::toggle_pin

    f) `create_snapshot(conn, view_id, items: &[RankingItemDto]) -> Result<()>`:
       - 构建 RankingSnapshot（ULID id, items_json, stats_json）
       - 调用 persistence_sqlite::ranking_repository::save_ranking_snapshot

    g) `get_rank_change(conn, view_id, current_items: &[RankingItemDto]) -> Result<Vec<Option<i32>>>`:
       - 获取上次 ranking_snapshot
       - 对比 items 的 rank 变化
       - 返回 Vec<Option<i32>> (Some(+N), Some(-N), None=首次)

            .join("geek-taste.db");
        std::fs::create_dir_all(db_path.parent().unwrap()).ok();
        let conn = rusqlite::Connection::open(&db_path).map_err(|e| e.to_string())?;
