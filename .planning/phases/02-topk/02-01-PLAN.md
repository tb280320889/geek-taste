       - ranking_snapshot_id: String

       - snapshot_at: String

       - repo_id: i64
       - full_name: String
       - rank: i32
       - score: f64
       - is_subscribed: bool

    6. `SnapshotStats` struct: total_count: i32, new_count: i32, changed_count: i32

    7. `MomentumScore` struct — 评分结果：
       - total: f64 (0.0-1.0)
       - star_delta_norm: f64
       - fork_delta_norm: f64
       - updated_recency_norm: f64

    8. 实现 Momentum 评分函数：
       ```rust
       pub fn compute_momentum(
           current_stars: i64, prev_stars: i64,
           current_forks: i64, prev_forks: i64,
           days_since_update: f64,
       ) -> MomentumScore
       ```
       - star_delta_norm = clamp((current_stars - prev_stars) / max_delta, 0, 1)
       - fork_delta_norm = clamp((current_forks - prev_forks) / max_delta, 0, 1)
       - updated_recency_norm = clamp(1.0 - days_since_update / 30.0, 0, 1)
       - total = 0.5 * star_delta_norm + 0.2 * fork_delta_norm + 0.3 * updated_recency_norm
       max_delta 参数化，默认 1000.0

    9. 更新 lib.rs：添加 `pub mod ranking;`
